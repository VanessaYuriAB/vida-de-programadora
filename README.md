## 🎵 Vida de Programadora — Capa de Playlist

Este é o **primeiro projeto educacional guiado desenvolvido como parte do Sprint 1 do**
**bootcamp de Desenvolvimento Web da TripleTen Brasil**. Consiste na criação de uma capa
de playlist estilizada, desenvolvida com HTML e CSS.

O objetivo do projeto foi aplicar, de forma prática e guiada, os **fundamentos
essenciais** **do desenvolvimento front-end**, como **estruturação de conteúdo com HTML**
e **estilização visual com CSS**.

[![GitHub Pages](https://img.shields.io/badge/GitHub_Pages-acesse_aqui-deeppink)](https://vanessayuriab.github.io/vida-de-programadora/)

## 📌 Contexto

O projeto simula um cenário real de startup, no qual seria necessário gerar capas de
playlists automaticamente, sem upload de imagens por parte do usuário, a fim de reduzir
carga no servidor.

A página desenvolvida exibe:

- Título da playlist
- Subtítulo
- Imagem de capa
- Ano e autor
- Descrição com link externo
- Estilos personalizados e layout centralizado

## 🚀 Tecnologias Utilizadas

- `HTML5`
  - Estrutura semântica
  - Uso correto de tags (`h1`, `h2`, `h3`, `p`, `a`, `img`, `div`)
  - Metadados e `favicon`

- `CSS3`
  - Seletores de elementos
  - Propriedades de tipografia
  - Gradiente de fundo
  - Centralização e espaçamentos
  - Manipulação de margens e paddings

## 🎯 O que foi praticado neste projeto

- Estrutura básica de um documento `HTML`
- Separação de responsabilidades entre `HTML` e `CSS`
- Criação e aplicação de regras `CSS` externas
- Uso de gradientes de fundo
- Organização visual em blocos (`div`)
- Centralização de elementos na tela
- Boas práticas iniciais de acessibilidade (`alt` em imagens)
- Links externos abrindo em nova aba

## 🖼️ Preview

O projeto gera uma capa quadrada (500 x 500px), com gradiente de fundo, tipografia
estilizada e imagem centralizada, simulando uma capa real de playlist.
